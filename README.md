## Getting Started

### Installation

1. Clone the repo
```sh
git clone https://github.com/joonacode/little-lemon-web.git
```
2. Install NPM packages
```sh
npm install
```
3. Run the project
```sh
npm run dev
```
4. Run thest
```sh
npm run test
```
